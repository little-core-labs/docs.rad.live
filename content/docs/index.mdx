# Rad TV Creator and Streaming Docs

&#x20;In these docs, you will learn how to:

* Create Video on [Rad TV](/docs/rad-tv-community-updates) and [Get Paid](/docs/getting-started-creators-studios-publishers/upload-and-monetize-videos) as a Creator Instantly using Smart Contracts on Base and Ethereum, with other blockchains coming soon
* Become an [Approved Creator](/docs/getting-started-creators-studios-publishers/get-approved-for-subscription-payments) and [Have Your Content Discoverable](/docs/getting-started-creators-studios-publishers/get-verified) On All Rad TV Apps and [Included in the Premium Subscription](/docs/getting-started-creators-studios-publishers/get-approved-for-subscription-payments)
* Develop applications using the Rad TV [API](/docs/getting-started-developer-api)
* [Create RSS Feeds](/docs/getting-started-creators-studios-publishers/publish-rss-from-outside-server)&#x20;
* [Launch a FAST Channel ](https://app.gitbook.com/o/nyGNujBIcfUP5OkgIwG4/s/6ZioINj2Z1FBpLjZIigo/~/changes/393/getting-started-creators-studios-publishers/distribute-your-fast-channel)on Rad TV Instantly
* Download Rad TV Apps and Watch Rad TV Content Across Devices including iOS, Apple TV, Apple Vision Pro, Android, Android TV, Quest, Amazon Fire, Web, PlayStation PS4 and PS5 consoles, PS VR, and PS VR2
* Subscribe to Premium or Premium+ to unlock incredibly RAD features and content or OWN your access via a first-of-its kind Stream Pass NFT (a subscription you can own and resell)&#x20;
* Set up [Sideloading](/docs/getting-started-subscribers-viewers/rad-tv-for-ps5-ps-vr2/sideloading) and UPnP on Local Network Servers like Jellyfin, Kodi, Emby, etc
* Add Content to Special Features like Sony Bravia Artwork
