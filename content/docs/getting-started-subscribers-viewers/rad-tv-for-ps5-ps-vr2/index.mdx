# Rad TV for PS5 / PS VR2

<Callout type="info">
Rad TV for PlayStation 5 and PS VR2 is actually two applications in one: a standard version and a VR version. The standard version on the PS5 is a 2D experience while the VR version has support for the PS VR2 and stereoscopic (3D) content.

Learn how to select a specific application version [here](./#how-to-choose-which-application-version).
</Callout>

## How to Install

Search "Rad TV" on the PlayStation store.

<Callout type="info">
Sony Interactive Entertainment removed streaming apps from its web-based store. You will need to go onto the device itself and find the application inside the store application, NOT on the website. Only games are listed on the PlayStation website.
</Callout>

## How to Select a Specific Application Version

*   ### When Launching and the PS VR2 Headset is Turned Off

    It will load directly into the standard version of the application.
*   ### When Launching and the PS VR2 Headset is Turned On

    You'll receive a dialog box asking which version: standard or VR of the application you would like to use.

<figure><img src="/images/PS5-Test-Kit-20240820-0114.png" alt="" width="375" /><figcaption></figcaption></figure>

*   ### If Already Launched and You Want to Switch Versions

    Simply power on or off the PS VR2 headset and you will get a dialog box asking whether you want to switch application versions.

We are working on adding a button on each version's sidebar that will allow you to easily switch between application versions.
</Callout>
